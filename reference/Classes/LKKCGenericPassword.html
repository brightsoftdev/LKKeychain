account

Account name. Shows up as “Account” in Keychain Access.

@property (nonatomic, retain) NSString *account

Discussion

This property corresponds to the kSecAttrAccount attribute.

This property is a primary key for generic password items. Modifying it may invalidate previously generated persistent IDs that refer to this item.

Class Methods

createPassword:service:account:

Create a new generic password.

+ (LKKCGenericPassword *)createPassword:(NSString *)password service:(NSString *)service account:(NSString *)account

Parameters

password

The password value.

service

The value for the service attribute. For application-specific passwords, use your application’s bundle id here.

account

The value for the account attribute. (A human-readable string of your choice.)

Discussion

You can use [LKKCKeychainItem addToKeychain:error:] to save the newly created password on a keychain.
