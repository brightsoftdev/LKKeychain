Class Methods

generatorWithKeychain:

Return a generator that creates keys in the specified keychain.

+ (LKKCKeyGenerator *)generatorWithKeychain:(LKKCKeychain *)keychain

Parameters

keychain

The keychain into which keys will be generated.

Return Value

A new key generator instance.

Discussion

The keychain may be nil if you don’t plan to store the generated keys permanently. If at all possible, please specify a keychain if you intend to store the key on one. Generating the key directly into a keychain is safer and more reliable than importing it later with [LKKCKeychainItem addToKeychain:error:].
