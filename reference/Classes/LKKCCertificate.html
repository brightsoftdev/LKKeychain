subjectKeyID

The subject key identifier of this certificate, if present.

@property (nonatomic, readonly) NSData *subjectKeyID

Discussion

The subject key identifier is an optional certificate extension that allows quickly identifying certificates that contain a particular public key. Its value may be derived from the public key itself (using a possibly truncated SHA-1 digest), or it may contain a unique value.

The value is nil if this certificate is not on a keychain.

This property corresponds to the kSecAttrSubjectKeyID attribute.
